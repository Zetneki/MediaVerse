About MediaVerse

Explore movies and series like never before.

MediaVerse is your personal companion to discover and track your favorite movies and series. Powered by TMDB API, it provides accurate and updated information for film enthusiasts.

This project was developed by Zoltai Zeteny Csongor as a way to learn web software development and to gain experience in modern web technologies.

GitHub: https://github.com/Zetneki

Key Features

  • Personal movie and series tracking
  • Custom library management
  • Search and discovery powered by TMDB API
  • Modern responsive UI with PrimeNG

Planned Improvements

  • Comments and ratings
  • Data statistics on profile
  • Book tracking powered by Goodreads API
  • Video game tracking powered by RAWG API
